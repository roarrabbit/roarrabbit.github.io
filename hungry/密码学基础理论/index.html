# 弃九法

  • 设式子：A*B=C
    1. 先将A,B,C分别弃掉9
      • 里面两两相加=9则去掉，9或0直接去掉
    2. 然后将A,B,C分别(mod9)得到a,b,c
    3. 判断(a*b)(mod9) == c，相等则正确

# 完全剩余系

  • 题：在Z_4中，属于[2]的整数是？
  • 答：在模4的剩余类中，属于集合[2]的数是：2、6、-30... - 模4的剩余类：{[0],[1],[2],[3]}
  • 定义：从模n中取一个数作为集合，称为模n的一个完全剩余系；
  • {0,1,2,···，n-1}称为模n的最小非负完全剩余系

# 互素

  • 互质是公约数只有1的两个整数
    • (a,b)=1；a与b的最大公约数为1

# 互质

  • 互质_是公约数只有1的两个整数

# 简化剩余系

  • 定义：模m的完全剩余系中与m互素的数组成的集合，称为模m的一个简化剩余系
  • 题：模12的简化剩余系是？
  • 答：1,5,7,11
    • 其中([1,5,7,11], 12)都=1

# 普莱费尔密码(playfair

  • 题：用playfair密码加密,明文hencookhot, 密钥为henfox,加密矩阵的第一行为____;加密矩阵的第二行为____;加密前分组为____（分组间用一个空格隔开）;密文为___

  • 答：这里做法不一定标准，仅供参考！具体以书为准

    • 先将明文分组：he nc ox ok ho tx

      • 分组有其规则，具体看下面

    • 然后画密钥矩阵

      e n f o
      a b c d
      i/j k l m
      q r s t
      v w y z

    • 加密矩阵的第一行为henfo

    • 加密矩阵的第二行为xabcd

    • 加密前分组为（分组间用一个空格隔开）he nc ox ok ho tx

    • 密文为enfbhdnmehpd

    • 分组规则

      1. 如果两个有字母相同的，则第二个用x代替
      2. 如果两个字母在同一行，用它右边的字母来代替，最右边字母用左边第一个字母代替，例如明文he，则密文为en。因为he同一行，所以用右边的字母代替
      3. 如果两个字母在同一列，用他下面的字母来代替，最底下的字母用该列第一个字母代替
      4. 其他的字母都用它同一行，另一个字母的同一列相交的字母代替；先行再列，例如明文nc，则密文为fb。将nc画十字，相交的为f和b，先n再c所以fb

# 四级线性反馈移位寄存器

  • 题：有一个四级线性反馈移位寄存器 f(x1,x2,x3,x4) =x1+x4, 其中初始值 (a1,a2,a3,a4)=(0011), 则输出的第7位为__7__, 则输出的前10位为__0011110101__

    • 将初始值写出，然后异或第一和第四位
    • 将异或的结果写右边，并将整体往移一位
    • 对比移位之后的四个数字
