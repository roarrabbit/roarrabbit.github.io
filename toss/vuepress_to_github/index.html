# 前言

博客没记错是从2019年的时候开始规划的，那个时候还是在本地使用wordpress搭建，后来知道可以用github page来搭建静态的网站就转去hexo+github page，直到最近经朋友介绍才知道vuepress，加上自己的博客已然太旧，所以近日打算将其进行更新改造！

# 前期准备

为避免不必要的报错，建议在cmd或powershell中使用命令

  • git
    $ git config --global user.name "bit"
$ git config --global user.email bit@qq.com
    1
    2
  • node.js的配置 node.js配置 (opens new window)
  • 安装yarn：npm install -g yarn
  • 初始化：yarn config set registry https://registry.npm.taobao.org
  • 新建文件夹以存放项目：mkdir vuepress-blog && cd vuepress-blog
  • 生成 vuepress-theme-hope：yarn vreate vuepress-theme-hope docs
    • 在执行的后续会需要你确认，默认无脑回车即可
  • 运行成功后即可在浏览器输入 localhost:8080/zh/
    • 若选择了不立即运行，可在之后输入 yarn docs:dev 进行本地启动运行
  • 这个时候其实已经完成了一个模板，你可以在次模板上进行修改操作，下面是我对模板的一些修改说明

# 文章架构说明

  • 具体的文件结构可以参考主题文档：文件结构介绍 | vuepress-theme-hope (opens new window)
  • 总的说
    • docs文章都放在这个文件夹里
      • readme.md：vuepress会自动生成为主页，即***.com/
      • links.md：访问的时候只需输入***.io/links即可
      • play_note：文件夹
        • readme.md：访问的时候只需输入***.io/play_note即可
        • vuepress.md：访问的时候只需输入***.io/play_note/vuepress即可
    • dist后续生成部署的时会放到这个文件夹里

# 更改语言为中文

温馨提示

  • 参考：多语言 (opens new window)
  • 此主题有中英文两个版本，若只是要中文，则需要设置./docs/.vuepress/config.js文件

  1. 找到config.js下locales语言配置

  2. 将键为"/zh/"的都删掉

  3. 将键为"/"的lang改为"zh-CN"

    locales: {
    "/": {
        // 设置需要的语言	
        lang: "zh-CN",
    },
},

# 评论服务

主题配备了Waline、Vssue 和 Valine，评论服务默认是使用Waline，但Waline需要服务器后台支持，这里我推荐使用Valine，你也可以参考这里 (opens new window)

  • 注册LeanCloud：Link (opens new window)
  • 登陆进去点击创建应用，完成后找到设置->应用Keys，将AppID和AppKey复制到comment中

      type: "valine",
      appId: "*****", // your appId
      appKey: "*****", // your appKey

# 部署静态页面并上传github page

  • 这样部署出来的文件需要后续手动部署上传，会有点不太方便

  • 这里默认你已经配置好了github仓库的内容，不会的话可以参考这里 (opens new window)

  • 手敲

    详细 
      # 生成静态文件
  yarn docs:build
  # 进入生成的文件夹
  cd /dist
  # 如果是发布到自定义域名
  # echo 'www.example.com' > CNAME
  git init
  git add -A
  git commit -m 'deploy'
  
  # 如果发布到 https://<USERNAME>.github.io
  git push -f git@github.com:<USERNAME>/<USERNAME>.github.io.git master
  
  # 如果发布到 https://<USERNAME>.github.io/<REPO>
  # git push -f git@github.com:<USERNAME>/<REPO>.git master:gh-pages
  
  # 把上面的 <USERNAME> 换成你自己的 Github 用户名，<REPO> 换成仓库名，比如我这里就是：
  #git push -f git@github.com:wtyqer/blog.git master:gh-pages
