windows10通知

RoarRabbit ... 2020-02-03 杂物间
  • Python
 小于 1 分钟

# 前言

利用python可以发送一条通知，如果只是单纯的通知比起弹窗会好很多！

# 动手

from win10toast import ToastNotifier as Tn
Tn = Tn()
while Tn.notification_active(): sleep(0.5)
# show_toast()中第一个参数为标题，第二个参数为正文，第三个参数为弹窗显示的时间
Tn.show_toast('我是标题', '我是正文', duration=2)
